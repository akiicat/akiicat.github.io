在 Ubuntu 上重新安裝 Kubernetes

2019-04-26
  • Kubernetes
  • Bare Metal
  • Installation
  • Kubernetes
  • Ubuntu

介紹

Kubeadm 有提供一個指令 reset，不過他只會將有關 Kubernetes 的東西刪除，像是 flannel、cni 的網路設定，則必須要手動刪除。

這裡使用的環境是：

  • Ubuntu 18.04
  • Kubernetes 1.14.1
  • Flannel 0.10.0
