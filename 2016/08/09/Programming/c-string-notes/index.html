strcpy 跟 strdup 的差別就在於：

  • strcpy 需要自己準備好字串的長度
  • strdup 則會自己幫你 alloc 一塊記憶體，但之後要自己 free 掉

切割字串 strtok

第一次 token 字串

擷取的字串為 p，切割字元為 _，最後把切割字串的起始位置存在 token 中 (表格中第二個箭頭的位置)

由左往右的箭頭分別為:

  • 第一個箭頭: string 起始位置
  • 第二個箭頭: token 指向的位置
  • 第三個箭頭: token 結尾
